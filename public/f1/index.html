Disclaimer: The GOAT score is calculated based on a weighted sum of various statistics including championships, wins, podiums, points, fastest laps, poles, grand prix participations, and seasons.

GOAT Score:
\((C×10)+(W×2)+P+(Pn×0.1)+F+Pl+(G×0.5)+S\)

\(C\) = Number of Championships
\(W\) = Number of Wins
\(P\) = Number of Podiums
\(Pn\) = Number of Points
\(F\) = Number of Fastest Laps
\(Pl\) = Number of Poles
\(G\) = Number of Grand Prix Participations
\(S\) = Number of Seasons
